# Soundy Design Playground

This folder contains standalone visual concepts built with plain HTML + CSS.

## Open in browser

- `designs/index.html` - gallery with links to all concepts
- `designs/01-sunrise-wave/index.html`
- `designs/02-midnight-glass/index.html`
- `designs/03-poster-noise/index.html`
- `designs/CONCEPT_PACK.md` - 6 structured text concept cards for future implementation

Each concept is intentionally different so you can compare style directions.
